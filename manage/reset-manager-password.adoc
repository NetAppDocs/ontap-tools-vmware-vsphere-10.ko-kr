= ONTAP 도구 관리자 암호를 재설정합니다
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ONTAP 도구 관리자 비밀번호를 잊어버린 경우 ONTAP tools for VMware vSphere 에서 생성된 재설정 토큰을 사용하여 관리자 액세스를 복원할 수 있습니다.

.단계
. 웹 브라우저를 열고 다음으로 이동합니다. `https://<ONTAPtoolsIP>:8443/virtualization/ui/` ONTAP 도구 관리자에 액세스합니다.
. 로그인 페이지에서 *비밀번호 재설정*을 선택하세요.
. ONTAP tools for VMware vSphere 사용하여 비밀번호 재설정 토큰을 생성합니다.
+
.. vCenter Server에 로그인하고 유지 관리 콘솔을 엽니다.
.. 입력하다 `2` *시스템 구성*을 선택하세요.
.. 입력하다 `3` *'maint' 사용자 비밀번호 변경*을 선택하세요.


. 비밀번호 재설정 대화 상자에서 재설정 토큰, 사용자 이름, 새 비밀번호를 입력합니다.
. 자격 증명을 업데이트하려면 *재설정*을 선택하세요.
. 새로운 비밀번호로 ONTAP 도구 관리자에 로그인하세요.
